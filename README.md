## Redux Express Small App

### Frontend Features
* Webpack
* Babel
* Sass
* React
* React Router
* React Portal
* Redux
* Redux Saga
* Redux Form

### Backend Features
* Express
* Sequelize
* Passport JWT
* Postgres

**Start Frontend**
```sh
npm run serve
```
**Build Frontend**
```sh
npm run build
```
**Start Backend**
```sh
npm run nodeserver
```
